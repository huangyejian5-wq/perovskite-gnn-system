# Perovskite GNN Prediction System

基于多尺度物理约束图神经网络的钙钛矿材料性能预测系统。当前仓库以钙钛矿材料性能预测为主线，围绕数据库整合、研究分析、性能预测、候选优化与 3D 可视化构建统一研究界面，而不是单纯的数据展示站。

## 项目定位

- 研究对象：钙钛矿材料，重点覆盖带隙、发光相关性质、稳定性与候选筛选
- 方法主线：多尺度建模 + 物理约束 + 多任务学习 + 候选优化
- 系统目标：把真实整理数据、模型分析和交互式可视化整合到同一个研究工作台

## 当前能力

- 多源数据库：集成实验带隙记录、补充文献、发光材料与器件记录、筛选候选和 Materials Project 主库
- 多尺度分析：支持组成、结构维度、数据域与尺度归属的统计分析
- 性能预测：提供钙钛矿组成输入、参考库匹配、预测结果展示与误差参考
- 候选优化：面向目标性能给出候选方案、研究先验和合成建议
- 3D 可视化：提供精选模板结构与真实公式联动浏览
- 方法说明：展示研究主线、数据来源和当前真实性边界

## 数据概览

当前仓库中已接入的核心数据模块包括：

- 实验带隙记录：133 条
- 筛选候选记录：3269 条
- 假金属拯救候选：1546 条
- Ground-truth 对齐摘要：4624 条
- 发光材料与器件记录：193 条
- Materials Project 衍生主库：6380 条

说明：

- 发光数据当前包含材料性质与器件性能两部分，已结构化接入 `PLQY`、`EQE`、`CIE`、`luminance`、`T50` 等字段
- 3D 可视化页采用“精选结构模板 + 真实公式联动”的展示方式，不将其表述为完整真实精修结构库
- 页面中论文目标能力与当前已完成真实接入的数据能力做了边界区分

## 页面组成

- `Home`：系统总览、研究主线和入口导航
- `Database`：多数据域检索、筛选、详情查看与多尺度标注
- `Analysis`：多尺度统计分析、补充文献层和验证基线展示
- `Prediction`：输入组成后给出预测结果与参考库映射
- `Optimization`：面向目标性能的候选优化与研究建议
- `Visualization`：3D 结构浏览、性质对比和模板联动
- `Methods`：项目背景、方法路线、数据来源和真实性边界说明

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动 Web 开发环境

```bash
npm start
```

默认会以 `HOST=0.0.0.0` 启动开发服务器，便于局域网演示和本机调试。

### 3. 启动桌面开发环境

```bash
npm run electron-dev
```

### 4. 构建生产版本

```bash
npm run build
```

### 5. 打包桌面应用

```bash
npm run dist
```

## 目录建议
